CSS弹性布局

1. 什么是弹性布局

通俗的讲，当一个页面当进行缩放的时候（ctrl+鼠标滚轮），其中的内容是否会随着变化？如果随着变化，那就是弹性设计。

2. 弹性布局基础知识

怎么引用flex布局

任何一个容器都可以指定为flex布局

.box{
    display:flex;
}

.box{
    display:inline-flex;
}

webkit内核的浏览器，必须加上-webkit前缀

.box{
    display:-webkit-flex;/*safari*/
    display:flex;
}

基本概念

  1. 容器：flex container，采用flex布局的元素，成为flex容器，简称容器。
  2. 项目：flex item，容器中包含的所有子元素自动成为容器的成员，称为flex项目。

  3. 轴：容器默认存在两根轴，水平的主轴（main axis）和垂直的交叉轴（cross axis）。主轴开始的位置叫main start，结束的位置叫main end，交叉轴开始的位置叫cross start，结束的位置叫cross end，项目默认沿主轴排列，单个项目占据的主轴空间叫做main size，占据的交叉空间叫cross size。

    3. 容器的属性

    1.flex-direction属性决定主轴的方向，即项目的排列方式

    .box{
    flex-direction:row|row-reverse|column|column-reverse
}
    • row:主轴为水平方向，起点在左端
    • row-reverse:主轴为水平方向，起点在右端
    • column：主轴为垂直方向，起点在上沿

    • column-reverse：主轴为垂直方向，起点在下沿
      2.flex-wrap属性定义如果一条轴线排不下，如何换行

      .box{

      flex-wrap:nowrap|wrap|wrap-reverse;

      }

    • nowrap:不换行
    • wrap：换行，第一行在上方

    • wrap-reverse：换行，第一行在下方
      3.flex-flow是1、2两个属性的简写形式

      .box{

      flex-flow:<fledirection>||<flex-wrap>;

      }

4.justify-content属性定义了项目在主轴上的对齐方式

    .box{
        justify-content:flex-start|flex-end|center|space-between|space-around;
    }
* flex-start:左对齐
* flex-end：右对齐
* center：居中
* space-between：两端对齐，项目之间的间隔相等
* space-ground：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。

5.align-items属性定义了项目在交叉轴上如何对齐

    .box{
        align-items:flex-start|flex-end|center|baseline|stretch;
    }
* flex-start：交叉轴的起点对齐
* flex-end：交叉轴的终点对齐
* center：交叉轴的中点对齐
* baseline：项目的第一行文字的基线对齐
* stretch：如果项目未设置高度或设为auto，将占满整个容器的高度。

6.align-content属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。

    .box{
        align-content:flex-start|flex-end|center|space-between|space-around|stretch
    }
* flex-start:与交叉轴的起点对齐。
* flex-end：与交叉轴的终点对齐。
* center：与交叉轴的中点对齐。
* space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
* space-around：每根轴线的两侧的间隔相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
* stretch：轴线占满整个交叉轴。

4. 项目的属性

1.order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0

.item{
    order:<integer>;
}

2.flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大

.item{
    flex-grow:<number>;
}

如果所有项目的flex-grow的属性都为1，则他们将等分剩余空间。如果一个项目的属性为2，其余都为1，则前者占据的剩余空间将比其他项多一倍。
3.flex-shrink属性定义了项目的缩小比例，默认为0，即如果空间不足，该项目将缩小

.item{
    flex-shrink:<number>;
}

如果所有项目的属性值都为1，当空间不足时将等比例缩小。如果一个为0，其他的为1，则空间不足时，前者不缩小。
4.flex-basic属性定义了在分配多余空间之前，项目占据的主轴空间。浏览器根据这个属性，计算主轴是否有多余空间。默认值为auto，即项目本来的大小。

.item{
    flex-basic:<length>|auto;
}

5.flex属性是2、3、4的简写，默认值为0,1，auto。后两个属性可选

.item{
    flex:none|[<'flex-grow'><'flex-shrink'>?||<'flex-basic'>]
}

改属性有两个快捷值：auto（1,1，auto）和none（0,0，auto）
建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。
6.align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素则等同于stretch。

.item{
    align-self:auto|flex-start|flex-end|center|baseline|stretch
}
