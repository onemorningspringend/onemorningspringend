#Markdown语法总结

##一、区块元素

###1.标题
Markdown支持两种标题的语法，类Setext和类atx形式
类Setext ：最高级标题“=” ，第二阶标题“-”，例如：

this is an h1

‘=============’

this is an h2

‘———————’

类atx：用1-6个“#”，对应标题1-6阶，例如：

/#h1

/##h2

/###h3

###2.区域引用 blockquotes
Markdown标记区块引用是使用 “>”，区域引用可以嵌套，只要保持”>”的数量对称就行，例如：

this is a blockquote

this is nested blockquote

###3.列表
Markdown支持有序列表和无序列表
无序列表是用“※”、“＋”或者“－”作为列表标记
有序列表则使用数字紧接着一个英文句点

###4.代码区块
和程序相关的写作或是标签语言原始码通常会有已经排好版的代码区块，通常这些区块我们是照着原来的样子显示，Markdown会用

和标签来吧代码区块包起来。要在Maarkdown中建立代码块很简单，只要简单的缩进4个空格或者是1个制表符就可以了。

###5.分隔线
你可以在一行中用三个以上的“※“”、“－”、“_”来建立一个分隔线，行内不能有其他东西

##二、区段元素

###1.链接
Markdown支持两种形式的链接语法 行内式和参考式，不管哪一种都是用“[ ]”来标记
行内式 只要在方括号后面紧接着圆括号并插入网址链接即可，例如：
this is an example
参考式 在链接文字的括号后面在街上另一个方括号，而在第二个方括号里面要填入用以辨识链接的标记，例如：
this is an example

链接内容的定义形式为：
·方括号，里面输入链接文字
·接着一个冒号
·接着一个以上的空格或制表符
·接着链接的网址
·选择性地接着title内容，可以用单引号、双引号或者是括弧包着

###2.强调
Markdown使用“※”和“_”作为标记强调字词的符号，被星号和底线保卫的字词会被转成用标签保卫，用两个星号或底线包起来的话，则会被转成。

###3.代码
如果要标记一小段代码，你可以用“ ‘ ’ ”把他包起来，反引号

###4.图片
Markdown使用一种和链接很相似的语法来标记图片，同样也是行内式和参考式
行内式 ：


参考式：
！Alt text

###5.其他
Markdown支持以比较简短的自动连接形式来处理网址和电子邮箱信箱，只要是用尖括号包起来，Markdown就会自动把他转成链接。一般网址的链接文字就和链接地址一样，例如：
http://example.com/

###6.反斜杠
Markdown 可以利用反斜杠来插入一些在语法中有其他意义的符号，例如：如果你想用星号在文字旁边文字旁边的方式来做出强调效果（但不用标签），你可以在星号的前面加上反斜杠。
